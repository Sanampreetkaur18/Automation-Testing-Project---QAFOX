QAFOX E-Commerce Website Automation Testing

Project Overview:
This project demonstrates automation testing of the QAFOX Demo E-Commerce Website, a publicly available demo website commonly used for learning and practicing software testing.
The project validates core user functionalities such as:

Registration
Login
Logout
Product Search
Product Comparison

Automation scripts are built using Selenium WebDriver with Java and TestNG, following the Page Object Model (POM) design pattern.
This project is intended for learning, practice, and portfolio demonstration purposes.

Objective
Verify critical user functionalities work as expected
Identify functional, UI, and usability defects
Ensure basic security behavior (e.g., password masking)
Provide portfolio-ready QA documentation

Features Tested
